Z_LASER_PROJECTOR - Documentation¶

Z_LASER_PROJECTOR is a Focused Technical Project (FTP) developed by FADA-CATEC, which belongs to a grant instrument within the ROSIN (ROS-Industrial) EU project. The aim of ROSIN is to progress towards the stated goal of create a collection of open-source, quality-assured software components for real-world robot applications.
