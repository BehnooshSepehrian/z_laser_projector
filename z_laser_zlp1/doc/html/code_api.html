message z_laser_projector/Line¶
Parameters:
  • group_name (string) – name of the projection elements group to which the line belongs
  • shape_id (string) – name of the line element to define
  • x (float64) – x position of the line’s starting point
  • y (float64) – y position of the line’s starting point
  • angle (float64) – line slope
  • length (float64) – line length

    service z_laser_projector/CoordinateSystem¶
    Request parameters:
     
    • name (string) – coordinate system name
    • distance (float64) – distance between the projector and the projection surface
    • P1 (geometry_msgs/Point) – x,y position of P1 point from projector reference system {P}
    • P2 (geometry_msgs/Point) – x,y position of P2 point from projector reference system {P}
    • P3 (geometry_msgs/Point) – x,y position of P3 point from projector reference system {P}
    • P4 (geometry_msgs/Point) – x,y position of P4 point from projector reference system {P}
    • T1 (geometry_msgs/Point) – x,y position of T1 point from user reference system {T}
    • resolution (float64) – resolution of user coordinate system {T}
    Response parameters:
     
    • T (geometry_msgs/Point[]) – return list of user reference points [T1,T2,T3,T4]
    • cs_created (bool) – return value response. True for success, False otherwise
    • message (string) – message response for error or success identification
